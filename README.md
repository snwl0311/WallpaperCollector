# WallpaperCollector
壁纸采集器：采集来源涵盖360壁纸库、必应首页壁纸以及金山词霸开放平台。


在线壁纸
========
特色：壁纸种类多、自动更新，自动采集、支持多分辨率、瀑布流加载、全站 ajax!

壁纸采集自 360壁纸库、必应首页的每日图片以及金山词霸开放平台

**本作品禁止任何形式的倒卖，转载请注明出处！**
